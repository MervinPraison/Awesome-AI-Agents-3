# Font source

`bricolage-grotesque-latin-700.woff2` is the Latin webfont subset for
[Bricolage Grotesque](https://fonts.google.com/specimen/Bricolage+Grotesque),
downloaded from Google Fonts on 2026-07-19.

The font is used only for the website's main display heading. It is licensed
under the SIL Open Font License 1.1; see `OFL.txt` in this directory.
