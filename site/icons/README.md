# Icon sources

The interface icons in this directory are vendored SVG assets rather than
hand-drawn paths or runtime CDN dependencies.

- `github.svg` is from [Simple Icons](https://simpleicons.org/) and is licensed
  under [CC0 1.0](https://github.com/simple-icons/simple-icons/blob/develop/LICENSE.md).
- `chevron-down.svg`, `external-link.svg`, `orbit.svg`, and `search.svg` are
  from [Lucide](https://lucide.dev/) and are licensed under the
  [ISC License](https://github.com/lucide-icons/lucide/blob/main/LICENSE).

Assets were downloaded from the projects' official GitHub repositories on
2026-07-19.
